# ⚡ DataWaster Pro v2.0

**DataWaster Pro** is an elite, high-performance web utility designed to consume internet data for testing, benchmarking, and bandwidth stress-testing. Built with a premium **Midnight Obsidian & Cyber Gold** aesthetic, it provides real-time metrics and extreme concurrency capabilities.

## ✨ Features

- **👑 Premium UI/UX**: A stunning "Terminal Command Center" design with glassmorphic cards and floating atmospheric particles.
- **📈 Smooth Visualizer**: A real-time bandwidth graph powered by a custom Bézier curve engine for fluid data visualization.
- **🚀 Extreme Mode**: Toggle "Unlimited Mode" to fire up to **60+ parallel threads**, saturating even high-speed Fiber and 5G connections.
- **📊 Precision Metrics**:
  - **Total Wasted**: Track cumulative data consumption (MB/GB).
  - **Live Speed**: High-precision readout in **MB/s**.
  - **Peak Speed**: Remembers your highest achieved throughput.
  - **Session Timer**: Accurate duration tracking.
- **📱 Fully Responsive**: Optimized for zero-scroll on desktop and a perfect touch-friendly grid on mobile devices.
- **⚡ Zero Dependencies**: Pure HTML5, Vanilla CSS3, and modern JavaScript.

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge).
- An internet connection (unmetered recommended!).

### Usage
1. Open `index.html` in your browser.
2. Adjust the **Intensity** (Number of Threads) to your preference.
3. Choose between **Normal** or **⚡ Extreme** mode.
4. Click **▶ START** to begin wasting data.
5. Watch the live metrics and the smooth bandwidth graph.
6. Click **■ STOP** to end the session.
